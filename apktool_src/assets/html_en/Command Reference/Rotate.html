Rotate

Operations:

Step 1: select objects to rotate; choose “Rotate” option;

Step 2: specify the rotation base point.

Step 3: adjust current point or specify rotation angle.

Step 4: adjust rotation angle or tap “✓” to confirm it.

Note: it provides copy function while rotating selected objects. You could rotate select objects or its copy by clicking the copy icon on the top right.
