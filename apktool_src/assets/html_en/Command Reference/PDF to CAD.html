Convert PDF to CAD

Function description:

It support to convert a PDF file to a CAD file.

Operations:

Step 1: open a PDF file;

Step 2: click “PDF to CAD” button on the bottom; it will upload and convert the PDF file automatically;

Step 3: view conversion progress by clicking the icon on the top right of the PDF page; or selecting the PDF to CAD option in the Home page by clicking +(on the right bottom of Home page).
The conversion results are saved at the following directory: My Cloud > PDF to CAD.

Conversion List:

You could find the following functions in conversion list:

1.View conversion statues.
The conversion result has three states: converting, succeeded, failed.

2.Jump to the My Cloud > PDF to CAD director:
Click on “Open” button and jump to My Cloud > PDF to DWG directory. All generated CAD files are saved under this directory.

3.Jump to the generated CAD file:
Clicking the arrow at the end of one PDF file, it displays a dialog box and inform conversion result. Click the “View now” button and jump to the generated CAD file which located in My Cloud > PDF to DWG directory.

4. Clear conversion record:
Long pressing on a PDF file and click “Clear” button to delete its conversion record.

5. Select one or more PDF files and add them to Conversion list:
You could select some PDF files from Recent, Local, and My Cloud file list, and add them to the PDF to CAD Conversion List.

6. Batch select and clear the converion record by click“Edit” button on the top right corner.
