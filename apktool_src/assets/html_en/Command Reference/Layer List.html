Layer List

It displays all layers in the list.

Turn on/off layer: Tap on one layer to turn it on or off.

Make Layer Current: click on the layer state icon to set the selected layer to be current layer.

Lock/unlock layer: click the lock/unlock icon in the layer list to lock/unlock objects on the selected layer.

Note: layers in external reference or frozen layers could not be set to be current layer.
