Useful Words

The Useful Words function is used to insert useful words to the current drawing conveniently.
Clicking on the Useful Words icon on the top right of screen, it displays words list.

Insert useful words
Clicking on one word and you could insert it to current drawing by specifying a point.

Edit useful words
Long pressing on a word, it pops up editing dialog box; if checked the “Lock text height”, you could insert words with a certain text height.

Manage useful words
Step 1: tap on the Manage icon;
Step 2: it is available to create new useful words, delete, edit ,or sort existing useful words.
