Arc length

Operations:
Step 1: click the “Measure” button and select the “Arc Length” tool, it displays the following panel by default:
Step 2: specify the start point of arc;
Step 3: specify the end point of arc;
Step 4: specify one point on the arc;
Step 5: it displays the following result on the panel: Arc radius, Total angle, Arc length, and Chord length. Long press in the value box and choose “Copy” option to copy measure result, click “×” behind value box or “×” behind command prompts to end command.
