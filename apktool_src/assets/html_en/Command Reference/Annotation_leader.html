Leader

Annotation → Leader   

Operation steps:

Step 1: in annotation list, choose “Leader” command;

Step 2: specify the start point of leader;

Step 3: specify the point to local Text box;

Step 4: enter text content. It is available to enter Mtext.

Step 5: end command by tapping “✓”on the top right corner.
