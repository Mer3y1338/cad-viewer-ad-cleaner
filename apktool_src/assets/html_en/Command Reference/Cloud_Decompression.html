Cloud Decompression

Function Description:

In My Cloud file list, you could decompress a compressed package file in local or on the cloud. Decompression in local means to download the compressed package file to local first, and then decompress it in the device's internal storage. Decompression on the cloud means to decompress it on the cloud, and all files decompressed are saved on the cloud.

Operation Steps:

Step 1: click a compressed package file in My Cloud file list;

Step 2: click "Cloud Decompression" on the bottom of dialogbox, it will decompress it automatically, and then it displays all decompressed files in the same location with compressed package.

Step 3: you could click the cloud decompression process icon to view the status of selected compressed package files on the top right of My Cloud.

Cloud Decompression file list:

1. View all decompression tasks;

2. Clear one task by long pressing.

3. Jump to decompressed position by clicking it.

4. Batch select and clear the decompression record by click“Edit” button on the top right corner.
