Insert Block

The Insert Block function is used to insert selected block to current drawing.
Operation steps:
Step1: start the Insert Block function in Toolbox;
Step 2: select a block in “My Block Library”;
Step 3: specify a point to local the block;
Step 4: justify the block location, or justify insertion angle, scale;
Step 5: confirm it by clicking on “√”， or tapping on blank. Clicking on “×”to canceling it.

If inserting a block with attribution, it will pop up an dialog box to edit attribution.

Other operations:
(1) It supports to choose blocks to insert from My Block Library or Current Drawing Library;
(2) It is available to create new blocks in My Block Library or Current Drawing Library, operations steps refer to New Block.
(3) Long pressing on a block in My Block Library, you could rename it or delete it;
(4) Lone pressing on a block in Current Drawing Library, you could add it to My Block Library.
