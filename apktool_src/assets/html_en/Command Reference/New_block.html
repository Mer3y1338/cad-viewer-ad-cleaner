New Block

The New block function is used to create a new block by selecting several objects from current drawing.

Operations:

Step 1: start Insert Block function.

Step 2: click “New Block” and specify block name in the pop-up.

Step 3: specify the base point.

Step 4: specify objects that combines the block. Click “√” to finish.

Note: if it contains attribute definition objects when specifying block objects, you could specify new attributes for the new block.
