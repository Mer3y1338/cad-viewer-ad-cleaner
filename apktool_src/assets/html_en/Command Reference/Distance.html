Distance

Operations:

Click the “Measure” button and select the “Distance” tool:
Step 1: specify the start point of measuring by clicking or dragging.
Step 2: specify the end point by clicking or dragging. It displays the following information between two specified points: Length, Angle, X Increment, and Y Increment.
Step 3: you could repeat the step 2 and step 3 to measure length of another segment.

Click the switch button and you could get a total length by specifying several points.
Step 1: specify points of measuring by clicking or dragging;
Step 2: if specified more than two points, it displays the total length of all segments.

Function instructions:
1. There is a new checkbox on the measurement panel, it is used to display the status of measurement result. If checked ,it means to keep measurement result on the drawing; otherwise the result disappears.
2. If you want to change measruement scale, you could click Scale icon on the panel.
3. If you want to change measurement precision, you could click Precision icon on the panel.
4. If you want to change text color, you could click the Color icon and choose another color.
5. If you want to change text height, you could click the Text Height icon to display the Text Height Settings dialog box. The text height is adaptive screen by default, and it could be edited by entering a new value or matching by selecting a existing text in the drawing.
6. Click “ √ ”to confirm the current input point and end the command.
7. Click “× ” to give up the current input point and end the command.
8. Long press in the value box and choose “Copy” option to copy measure result.

Note:
1. After recording the result, you can export Detailed Result in“Result” to view the length of each section.
2. For the results labeled on the graph, they can be viewed in“Result Statistics” and exported to Excel.
3. If you have mastered how to snap accurately by long pressing on the screen, you could turn off the settings of “Support to adjust point position after measuring”.
