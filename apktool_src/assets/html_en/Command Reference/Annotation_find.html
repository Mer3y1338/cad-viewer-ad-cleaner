Find Annotation

Toolbox → Find Annotation   

Function description: it is used to find annotations by certain filter. Filter criteria include annotation type, time range, and drawing range.

Step 1: start Find Annotation function;

Step 2: Set the filter criteria. In the initial state, all types of annotations are found by default. In the full time range, all creators, and the full map range, click "Find" to start searching all annotations; You can set annotation type, time range, creator and drawing range;

Step 3: after clicking Find button, it displays result list, and hide the filter panel; you could click Filter icon on the top right to display filter panel again. The first result displays in the current view;

Step 4: click on another result item in the list, the view changes and locates the location of selected item.
