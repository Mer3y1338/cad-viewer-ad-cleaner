Linear Dimension

Create a Linear dimension:
Operations:
Step 1: start Aligned dimension command;
Step 2: specify the first extension line origin;
Step 3: specify the second extension line origin;
Step 4: specify dimension line location; you could drag the moving icon to adjust the dimension line location. It is available to switch between horizontal and vertical direction.
Step 5: tap keyword “✓” or tap on blank to finish.
