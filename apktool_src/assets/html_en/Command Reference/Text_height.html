Text Height

Function description: it is used to modify the text height.

Operation steps:
Step 1: select one or more text, such as single text, Mtext, Attribute, Dimension;
Step 2: click “Text Height” function;
Step 3: in the “Edit Text Height” dialog box, it displays the text height of selected text; the height could be edited by entering a new value or matching by selecting a existing text in the drawing.
