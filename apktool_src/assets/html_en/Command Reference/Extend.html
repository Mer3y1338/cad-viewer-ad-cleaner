Extend

There are two work modes for Extend command ,Quick extend mode and Select edges mode. The Quick extend mode is default mode for Extend command.

Quick extend mode:


In Quick extend mode, the selected objects will be extended.
You could select an object by tapping on it or draw lines intersected with it.
You could tap Settings button and switch to Select edges mode.

Select edges mode:


Operation steps:
Step1: select boundary by tapping on objects or crossing selecting;
Step2: after selecting boundary, it display: N objects selected, you could click “Finish” button in the panel to next step;
Step 3: select object to extend according to command prompt; and the selected objects will be extended.

Extend Settings:

Extend mode:
Tap on one mode, it will switch immediately.
Edge extension mode:
It is controlled by EDGEMODE system variable.
