Facade

Function Description:
The Facade measurement command is used to calculate facade of building. The Facade is product of length and height. The length of building could be measured by specifying points, or by entering the value.

Operations:
Step 1: click “Facade” command to start it. There is a new checkbox on the measurement panel, it is used to display the status of measurement result. If checked ,it means to keep measurement result on the drawing; otherwise the result disappears.
Step 2:specify the length, you could specify points or enter the value.
Step 3: specify the height value.
Step 4: click “Calculate” to get lateral area.
Tips:
(1) If the checkbox is checked, it means you want to mark measurement result, when you click “ √ ”,it will keep the area result on the drawing and end the command. Click “× ” to give up the current measurement result and end the command. During operations, each area measurement result can be named for easy to distinguish.
If the checkbox is unchecked, whether click “ √ ” or “× ”, the measurement result will not be marked on the drawing.
If you want to change text color, you could click the Color icon and choose another color.
If you want to change text height, you could click the Text Height icon to display the Text Height Settings dialog box. The text height is adaptive screen by default, and it could be edited by entering a new value or matching by selecting a existing text in the drawing.
(2) If you want to change measruement scale, you could click Scale icon on the panel.
(3) If you want to change text color, you could click the Color icon and choose another color.
(4) If you want to change text height, you could click the Text Height icon to display the Text Height Settings dialog box. The text height is adaptive screen by default, and it could be edited by entering a new value or matching by selecting a existing text in the drawing.
(5) it supports to display area results in meters (m) in settings.
(6) long press in the value box and choose “Copy” option to copy measurement result.

Note:
(1) After recording the result, you can export Detailed Result in“Result” to view the length of each section.
(2) For the results labeled on the graph, they can be viewed in“Result Statistics” and exported to Excel.
(3) If you have mastered how to snap accurately by long pressing on the screen, you could turn off the settings of “Support to adjust point position after measuring”.
