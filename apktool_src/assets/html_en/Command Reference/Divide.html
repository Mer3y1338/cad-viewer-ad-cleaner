Divide

The Divide function is used to create evenly spaced point objects along the length or perimeter of an object.

Operations:

Step 1: click “Divide” function in “Draw” menu.

Step 2: specify the object to be divided; you could select a 2D polyline, a 3D polyline, an arc, a circle, an ellipse, an ellipse arc, a line, a polyline, a spline.

Step 3: specify the number of line segments to be divided and click “OK” to finish.
