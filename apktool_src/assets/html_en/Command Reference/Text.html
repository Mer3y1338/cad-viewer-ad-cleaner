Text

Operations:

Step 1: click the “Draw” button and select the “Text” command.

Step 2: Specify a point to determine the text position by clicking or dragging in drawing area.

Step 3: you could enter Mtext and return it in popping box. If you want to change text height, you could click the Text Height icon to display the Text Height Settings dialog box. The text height is adaptive screen by default, and it could be edited by entering a new value or matching by selecting a existing text in the drawing.

Step 4: tap “✓” to confirm it.
