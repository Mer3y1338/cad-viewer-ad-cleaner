Multileader

Operations:

Step 1: click the “Draw” button and select the “Multileader” command.

Step 2: specify the start point of leader by clicking or dragging in drawing area;

Step 3: adjust the start point or specify the end point of leader;

Step 4: input the content in the popping box; if you want to change text height, you could click the Text Height icon to display the Text Height Settings dialog box. The text height is adaptive screen by default, and it could be edited by entering a new value or matching by selecting a existing text in the drawing.

Step 5: tap “✓” to confirm it.

Edit:
you could edit created Multileader. Tapping on the leader and arrowhead, you could stretch the pinch to adjust its direction and size; tapping on the text, you could select “Edit Text” option and modify text in popping box.
