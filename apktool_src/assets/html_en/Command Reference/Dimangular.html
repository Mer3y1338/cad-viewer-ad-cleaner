Angular Dimension

Create an Angular dimension:
Operations:
Step 1: start Angular dimension command;
Step 2: specify the first angle endpoint;
Step 3: specify the angle vertex;
Step 4: specify the second angle endpoint;
Step 5: specify dimension arc line location; you could drag the arrow to adjust the dimension line location;
Step 6:tap keyword “✓” or tap on blank to finish.
