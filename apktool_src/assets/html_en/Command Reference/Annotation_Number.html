Annotation Number

The Annotation Number function is used to number the selected number or letter.

Operations:

Step 1: in annotation list, choose “Annotation Number” command;

Step 2: select the number or letter that you want to number in the drawing;

Step 3:check the number or letter that you want to number in the dialog box;

Step 4:click “OK” , and you can see the new generated number or letter. Drag the Move icon to adjust its position.

Step 5:click “Next” to generate the next result.

Step 6:click “ √ ”to keep all results and end the command; click “× ” to give up the last one result and end the command.
