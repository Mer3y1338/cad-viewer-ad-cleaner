Batch Measurement

Function Description:
The Batch Measurement command is used to measure the total length of a group line segments with some same attributes by specifying one line segment of them.

Operations:
Step 1: click the “Measure” button and select the “Batch” tool.
Step 2: specify one line segment.
In the view area, all line segments that meet the conditions are highlighted, while those that do not meet the conditions are dimmed. The total length value can be seen on the panel. If you need to count other line segments, you can click “Reselect” to select the line segments you want to count again.
