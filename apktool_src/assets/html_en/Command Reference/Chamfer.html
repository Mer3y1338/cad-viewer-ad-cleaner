Chamfer

There are two ways to create a chamfer, specifying two chamfer distances, or specifying one chamfer distance and one chamfer angle.

1. specifying two chamfer distances

Step 1:enter Dist1 and Dist2 in the panel;
Step 2: specify the first line;
Step 3: specify the second line;

2. specifying distance and angle

Step 1: enter chamfer distance and angle in the panel;
Step 2: specify the first line;
Step 3: specify the second line;

Chamfer Settings

Polyline:
Chamfer the whole 2D polyline.
Trim:
This option is used to control whether to trim specified line to the end point of chamfer.
Multiple:
Create chamfers for more than one set of objects.
