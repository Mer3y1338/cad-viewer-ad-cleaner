Find

It provides Text Find and Replace function in Edit Mode, and only Text Find function in View Mode.
Operation steps for finding text:
Step 1 : start Find command;
Step 2 : enter keywords in the search bar;
Step 3 : tapping on “Find”icon or tapping on one item in the list, it jumps to another page with highlight result displaying in the center.
There are some bubbles in the drawing to show the locations of keywords.
Tapping on one bubble, it jumps to display its highlight result in the center.

Operation steps for replacing text:
In Edit Mode, you could replace text in the following way:
Step 1 : enter keywords in the search bar, and tap on “Replace”option in Settings menu on the top right corner.

Step 2 : enter the content for replacement;


Step 3 : tapping on “Replace” button or tapping on one item in the list, it jumps to another page with highlight result displaying in the center.
Tapping on “Replace” button to perform text replacement.
Step 4: tapping on “Replace All” button to replace all in order of list.
Note: replacement for text in external references or locked layers are not available; it can be found in full view, current view, rectangular window, or shaped window range.
