Statistic Result

Function Description:
Now you are available to keep your measurement result on the drawing, and you could see all results in Statistic Result function. The result could be viewed or exported by sorts, such as  Distance, Area.

Operation steps:
Step1: start Statistic Result function;
Step2: specify statistic region;
It should be all measurement results in the drawing with all kinds of color by default.
You could specify statistic region（full view, rectangular window, or shaped window range）or specify colors.
Step 3: the corresponding results display in the list.
Step 4: click “Export” to export current sort of result to Excel, and you could view the Excel file in Recent file list.

Note: You could name each measurement result by clicking name in the sheet of Area or Side area。
