Dimension Settings

Command description: 
It is used to modify settings of the selected dimension object.

After selected the dimension object, tapping on the "Parameter edit" icon, and you could see the Dimension Settings dialog box. In this dialog box, It is available to modify the following settings, such as text size, arrow style, scale, and suffix.

Text size: modify text size by the following two ways.
Method 1: drag the slider to make the text bigger or smaller.
Method 2: tap "Word height match", and select a text in the drawing to make the dimension text size the same as selected text.
Arrow style: modify default arrow style.
The default arrow style is Architectural tick, and it is available to change to be Closed filled or Oblique.
Scale: modify scale of length.
The default scale is 1.
Suffix: modify default dimension suffix.
The default suffix is null.

In the drawing area, tap "More" icon on the top bar, and then tap on "Settings", select "Dimension Settings" option, and it is available to modify arrow style and suffix of dimension objects. The modified dimension settings will be applied on new created dimension objects.
