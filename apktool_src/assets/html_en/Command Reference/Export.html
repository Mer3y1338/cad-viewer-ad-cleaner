Export

The current drawing could be exported to PDF format，or picture format, such as BMP, JPG, and PNG.

Steps for exporting to a one page PDF:

1. Clicking “Export” icon in the more information menu on the top right，and select “Export to a one page PDF”；
2. Enter file name；
3. Specify the file saving location;
4. Select paper size or default；
5. Select paper orientation or default；
6. Select color or default：
7. Select export extent, default by current view;
8. Specify whether it contains layer information;
9. Specify whether it displays annotations;
10. Check “Fit to paper” or default;
11. If checked“Fit to paper” ,the export scale is a fixed value; otherwise the export scale could be customized;
12. Click “Export”on the top right corner to finish.

Steps for exporting to a multipage PDF:

1. Clicking “Export” icon in the more information menu on the top right，and select “Export to a multipage PDF”；
2. Enter file name；
3. Specify the file saving location;
4. Select paper size or default；
5. Select paper orientation or default；
6. Select color or default：
7. Specify whether it contains layer information;
8. Specify whether it displays annotations;
9. The export extent is fitting to paper by default;
10. Click “Next” to specify upper left corner point and lower right corner points for each page;
11. Click “√”on the top right corner to finish.

Steps for exporting to image:

1. Clicking “Export” icon in the more information menu on the top right，and select “Export to Image”;
2. It displays a rectangle in the interface, you can drag the arrow on the sides of rectangle to define picture boundary;
3. After adjusting the region click “✓” to continue;
4. Enter file name;
5. Specify the file saving location;
6. Select export format: BMP, JPG, or PNG;
7. Select color depth:
8. Specify whether it displays annotations when exported pictures;
9. Click “Export”on the top right corner to finish.

Location for saving exported files:

1. For Local drawings:
When exporting a PDF/image from local drawing, the exported files are saved in the same file folder as its drawing.
2. For Drawings in My Cloud or Cloud storage:
When exporting a PDF/image from a file in My Cloud or Cloud storage, the exported files are saved in Export folder under the root director.
