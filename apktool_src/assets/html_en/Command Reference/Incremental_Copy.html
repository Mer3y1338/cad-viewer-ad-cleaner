Incremental Copy

Command description: 
when copy a selected text, it could be copied according to a certain rules by specifying a number or alpha. It is a very helpful feature when creating incremental code or sequence number.

Operation steps:
Step 1: tap “Incremental Copy” function in Toolbox;
Step 2: specify a text according to command prompt;
Step 3: check the number or alphabet in the dialog box;
Step 4: tap “OK” to generate incremental copy text, and drag the Move icon to adjust its location;
Step 5: tap “Next” icon on the panel to generate the next incremental copy text.
Step 6: tap “×” to cancel the last generated text and end the command; tap “√” to conform the last generated text and end the command.
