  • View 3D models

On the basis of supporting to view 3D RVT files, the DWG FastView app newly supports to view mainstream 3D software formats, including Solidworks, Creo, NX, CATIA, Inventor, SolidEdge, and so on.
The list of supported formats is as follows:
.rvt, .3dxml, .sat, .sab, .model, .CATPart, .cgr, .ifc, .igs, .iges, .ipt, .iam, .jt, .prt, .obj, .x_t, .x_b, .asm, .prt.*, .asm.*, .3dm, .par, .asm, .psm, .sldprt, .sldasm, .stp, .step, .stl, .vda, .CATProduct, .gltf., .rfa, .skp

Note: Because the .bin file is the dependecy file of .gltf. To make sure the model displaying completely, it is suggested to put the.gltf file and its dependency files in the same folder, upload them to My Cloud by DWG FastView for Web, and then view them on the mobile phone.

The way to view a 3D model:

(1) Open a 3D model from Weichat, QQ, or other third party applications: click “DWG FastView” from the third party applications list, and then view it after the process of converting.

(2) View a 3D model in My Cloud: upload the 3D model to your account of My Cloud, and then click it to view after the process of converting.

Note: It may take a long time for converting 3D models, during the time you could view other drawings by clicking on “Convert in background” button. You could view them in Recent or My Cloud file list after successfully converted.

Top tools:

Previous View: Display the previous view.
Next View: Display the next view.
Reset: Restore Default view.
Full View: Display full view.
Full Screen: Display views in full screen.

Bottom tools:

Visual style: provide Shaded with edges, Shaded, Wireframe, Hidden visual styles; and addtional provide X-ray visual style for BIM.
View: support 10 kinds of views, they are Top view, Bottom view, Left view, Right view, Front view, Back view, SW Isometric, SE Isometric, NE Isometric, NW Isometric.
Feature Manager: Display all features for 3D models.
Display PMI: Display PMI of current model.
Measure:Support various kinds of measurement fucntions.
(1)Distance: Intelligent Distance, Point to Point, Point to Line, Point to Plane, Line to Line, Line to Plane, Plane to Plane.
(2)Angle: Intelligent Angle, Angle between lines, Angle between planes, Angle between a line and a plane.
(3)Circle related: Radius, Diameter, Arc length.
(4)Attribute related: Point, Line, Plane.
Measurement method:
After starting measurement function, When the finger moves near the object to be measured (point, line, surface), the measured object will be highlighted. At this time, lifting the finger selects the measured object.
Hide and delete menasurement results:
Uncheck the 'Display' option to hide all measurement results.
Click the X on the right side of the measurement result to delete the measurement result.
Click the 'Clear' option to clear all measurement results in the current model.
Section: View section plane for 3D models.
Explode: Create explosion view for 3D models.
Match Color: Automatically match color for each parts.
Settings: Set backgroud color, and set default view directions.
