My Cloud

1. Files

My Cloud is a series of cloud services provide by DWG FastView. Once you register a My Cloud account and log in, you could manage your drawings in My Cloud on mobile, on PC, or on Web.

Open a file:
Tap on the file to open it.

Filter files:

Filter: filter files by formats.

Sort: sort files by name, time, size, or type.

New Folder: create a new file folder in current page.

Search: Search files in current page.

Thumbnail: files display in thumbnail mode.

List: files display in list mode.

Path: display current file path. Tap on folder name in file path to open this file folder.

Operation one file:

Long pressing on one file, it displays all related operations for it.
You could also find these functions by tapping on Detail icon on the right of file name.
If selected a 3D CAD file, it provides “Convert” function in addition.
If selected a compressed package file, it provides “Decompression” function in addition. You could decompress it in local or on the cloud. Decompression in local means to download the compressed package file to local first, and then decompress it in the device's internal storage. Decompression on the cloud means to decompress it on the cloud, and all files decompressed are saved on the cloud. Details refer to "Cloud Decompression".

Batch operations:
Tap “Edit”on the top right corner of this page, you could select files in current page and do the following related batch operations:

Synchronize: when there is something different between the files in local and in server, you cloud tap “Synchronize” to keep them the same. The newer version will automatically replace the older version.
Operation Steps:
(1) Tap “Edit” on the top right corner of this page;
(2) Select files to synchronize;
(3) Tap “Synchronize”;
(4) Popping up progress dialog box, when it goes to 100%, synchronizing files finished. You could also cancel it by tapping “Cancel”.

Move：you could move one or more files to any other location.
Operation steps:
(1) Tap “Edit” on the top right corner of this page;
(2) Select files to move;
(3) Tap “Move”;
(4) Select the location(you could tap “New Folder” to create a new folder if necessary);
(5) Tap “Ok” to finish moving.

Delete: delete selected files.

When a 3D model selected, it is available to convert it for viewing. If you click Convert icon on the bottom after selected a 3D model that has been converted successfully before, it will be converted it for twice.
Convert: convert the selected 3D model.

Tap “Edit” again to cancel selection.

Refresh the file list:

Swipe down the current page to refresh the file list.

File status and their tapping operations ：

Need to synchronize: it means there is something different between files in local and in My Cloud (the difference may be occurred in drawing objects or annotations, or both of them). Tap the red button to synchronize files, the newer version will replace the older one. Tap on file name to open it without synchronizing.
Synchronized: The file is already for viewing. Tap file name to open it.
Need to download: it means the file has not downloaded yet. Tap the file name to download it. After downloading, the file automatically opened, and the gray button turns to be green button.
Confilict: files in local and in My Cloud have made some modifications separately, and they should be synchronized. Tapping on file name to open the local file; tapping on yellow button and confirming to synchronize, the local file will be renamed as original name_1, and then uploaded to My Cloud in the same localtion.
Need to convert: 3D models need to be converted for viewing. Tap file name to convert it.
Conversion failed: The 3D model has been converted failed before. It is not available for viewing.
Converting: The 3D model is converting.

3. Conflicted File

When is there a conflict occurred?
(1) you have logged in your My Cloud in A device and B device via the same account.
(2) you have downloaded the same drawing to your two devices separately.
(3) You have modified the drawing and saved it in A device, and then you synchronize it to My Cloud.
(4) You have modified the drawing and save it in B device, but you have not synchronize it to My Cloud.
(5) In B device, if you pull-down to refresh or reentry your account, the drawing status becomes to be red. Tap the red button, it prompts a conflict between files in local and in My Cloud.

How to solve the conflict?
When it prompts a conflict between files in local and in My Cloud, you could tap “Ok” to continue. The file modified in A device(which has already been uploaded to My Cloud) will be downloaded to B device, and the file modified in B device will be renamed as original name_1, and then uploaded to My Cloud in the same localtion.

4. Share files in My Cloud

Share steps:

Step 1: Long press on file name, and select "Share" button;
Step 2: in the File share dialog box, you could set Link validity period and visibility of annotataion, and then confirm to share it.
Step 3: you could select a way to send the sharing information.

Receive steps:

Step 1: when you receive a file sent by your friend, according to the information, you could tap on "Copy" button;
Step 2: and then open DWG FastView, then you could see a popup dialog box, it displays that someone has sent you this file, and you could select to "Save to My Cloud" or "Open" it directly.
