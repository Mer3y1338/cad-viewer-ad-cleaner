Historic Version

Prerequisites

You need to have a project (self-created project or joined project) and have already uploaded drawings.

Historic Version

[Step 1]
Click "Project" to enter the project list, select a project.

[Step 2]
Go to the "Drawings Management" section of the project,click to view the details of one drawing and enter the drawing details page.

[Step 3]
On the drawing details page, click History Version to enter the History Version page, in which you can view, download, rollback and delete the history version.

Note:In the project, the user who is responsible for assigning the "View/Save/Upload New Version" privilege can use the view and rollback functions. The person in charge can use the view, rollback and delete functions. Users with View permission cannot use this function.
