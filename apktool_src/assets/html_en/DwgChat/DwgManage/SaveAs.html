Save as

Prerequisites

You need to have a project (created by yourself) or a friend who has uploaded a drawing.

Save as

[Step 1]
Click "Project" to enter the project list, select a project.

[Step 2]
Go to the "Drawings Management" section of the project,click to view the details of one drawing and enter the drawing details page.

[Step 3]
Click "Save as" in the detail page to download and save drawings.

Note: In the project, only users who are responsible for assigning "View/Save as/Upload New Version" privileges can use it, and users with viewing privileges can not use the save function. In the private chat, both people can use the save function.
Users can view the details of the drawing through the Homepage, Cloud List, Local Drawing List, and Project List. Clicking "Save As" allows users to save the drawing as a different file.
