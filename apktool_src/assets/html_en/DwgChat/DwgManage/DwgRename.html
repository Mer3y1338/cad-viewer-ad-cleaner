Rename Drawing

Prerequisites

You need to have projects (self-created projects and joined projects) and have uploaded drawings.

Rename Drawing

[Step 1]
Click "Project" to enter the project list, select a project.

[Step 2]
Go to the "Drawings Management" section of the project,click to view the details of one drawing and enter the drawing details page.

[Step 3]
Click "Rename" on the details page and enter the new name of the drawing to rename it.

Users can view the details of the drawings through the Homepage, Cloud List, Local Drawing List, and Project List. Clicking "Rename" on the details page allows users to rename the drawing.

Note:Rename drawing can only be used by project creator (person in charge), project members can't use this function.
