Move Drawing

Prerequisites

You need to have projects (self-created projects and joined projects) and have uploaded drawings.

Move Drawing

[Step 1]
Click "Project" to enter the project list, select a project.

[Step 2]
Go to the "Drawings Management" section of the project,click to view the details of one drawing and enter the drawing details page.

[Step 3]
Click "Move Drawing" on the detail page, select a target directory to move the drawing.

Note:Moving drawings can only be used by the project creator (person in charge), project members can't use this function. Users can view the details of the drawings through the Homepage, Cloud List, Local Drawing List, and Project List. Clicking "Move" on the details page allows users to move the drawing.
