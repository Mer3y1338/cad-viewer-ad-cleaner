Upload new version

Prerequisites

You need to have a project (self-created project or joined project) and have already uploaded drawings.

Upload new version

[Step 1]
Click "Project" to enter the project list, select a project.

[Step 2]
Go to the "Drawings Management" section of the project,click to view the details of one drawing and enter the drawing details page.

[Step 3]
On the drawing details page, click the Upload New Version button to bring up the Select Drawing Source menu.

[Step 4]
Select the new version of the current drawing to upload the new version.

Note:Users in the project who need to be assigned the "View\Save As\Upload New Version" permission by the person in charge can upload the new version. Users with view permission cannot use this function. This function is also not supported in private chat.
