Create Folder

Prerequisites

You need to have projects (self-created projects and joined projects) and have uploaded drawings.

Create Folder

[Step 1]
Click "Project" in the navigation to enter the project list.

[Step 2]
Select a project and go to Drawing Management in the project details.

[Step 3]
In Drawing Management, click the "+" in the upper right corner and enter a folder name to create a folder.

Note

1. You can only create folders in the project, not in private chat.
2. Only the person in charge has the permission to create folders, and members do not have the permission to create folders.
