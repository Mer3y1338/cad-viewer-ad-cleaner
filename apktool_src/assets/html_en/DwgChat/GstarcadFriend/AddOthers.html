More ways to add friends

Procedure

1. Enter the program, click "Contacts icon in the upper right corner" - "Add Friends icon in the upper right corner" - "More Ways".
2. in the pop-up menu, select an APP, and then select friends in the APP, send a friend request.
3.Wait for your friend to pass the verification

Note

1. the friend you want to add needs to download and install GstarCAD for Mobile.
2. you want to add friends need to log in to see the king, and then click on the link you send!
