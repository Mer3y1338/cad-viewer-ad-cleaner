Project Introduction
1.What can the Project do?

The Project function module is a feature of CAD Viewer that is used to improve the efficiency of drawing communication among project teams and enable multi-user collaborative viewing. It consists of Drawing management, Personnel management, Instant messaging, DwgChat.

Drawing Management:The drawings in the Drawing management are uploaded by users to the project. After one person modifies the drawing and synchronizes it, others can view the latest modifications anytime and anywhere through the synchronization function. At the same time, the historical version function is provided to facilitate viewing multiple versions of a drawing.

Personnel Management:Project members can add other users to the project, and the project leader can assign different permissions to each project member to improve the security of drawings.

Instant Messaging:You can send and receive normal text, voice, files and pictures. In addition to this you can also send CAD drawings directly in a chat session.

DwgChatDwgChat allows multiple people to view a drawing remotely, and one person's operations on the drawing can be displayed simultaneously on the terminals of other participants in real time. Meanwhile, it is possible to receive and send messages while viewing or operating the drawing.

2. What equipment do I need to use the Project functions?

The Project function is commonly used on multiple platforms: Android devices, iOS devices, and PCs. Please upgrade CAD Viewer to the latest version before using it.

3. Is the project function free of charge?

Only registered users can use the project features, ordinary members can use all the core features of the project for free. At the same time, we provide Ad-block membership, Premium membership, Super membership and Company membership for users who have more needs to use the program, and they can enjoy more services after upgrading.
