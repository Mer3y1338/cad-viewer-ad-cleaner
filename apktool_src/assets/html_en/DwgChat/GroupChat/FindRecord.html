Search Chat History

Prerequisites

You need to have created a project by creating a project or have joined a project. "Search Chat History" can be viewed by both the project creator (the person in charge) and the members of the project.

Search Chat History

[Step 1]
Click "Project" --> Select Project --> "Project Details" --> "More" in the upper right corner to enter the project settings page.

[Step 2]
Click "Search Chat History" on the item settings page to enter the Search Chat History page.

[Step 3]
Click the "Search Chat History" section on the project settings screen to access.Go to the Search Chat History page. Enter a keyword in the input box to find the content of the relevant Chat History.
