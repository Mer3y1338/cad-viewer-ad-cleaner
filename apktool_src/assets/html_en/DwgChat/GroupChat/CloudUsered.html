Project Space Usage

Prerequisites

You need to have created a project by creating a project or have joined a project. The project space usage can be viewed by both the project creator (the person in charge) and the members of the project.

Project Space Usage

[Step 1]
Click "Project" --> Select Project --> "Project Details" --> "More" in the upper right corner to enter the project settings page.

[Step 2]
The "Project space usage" on the project settings screen displays the space usage of all drawings in the project.

Note:For ordinary projects, this value represents the size of the cloud storage space occupied by the responsible person for the current project.
