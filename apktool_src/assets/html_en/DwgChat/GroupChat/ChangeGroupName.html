Modify Project Name

Prerequisites

You need to have created the project by creating the project. The Modify Project Name feature is only available to the

Modify Project Name

[Step 1]
Click "Project" --> Select Project --> "Project Details" --> "More" in the upper right corner to enter the project settings page.

[Step 2]
Click the "Project Name" on the project settings page to modify the project name.

[Step 3]
Enter the new project name on the Modify Project Name page, and then click Finish to complete the modification.

Note: Only the person in charge has the right to modify the project name.
