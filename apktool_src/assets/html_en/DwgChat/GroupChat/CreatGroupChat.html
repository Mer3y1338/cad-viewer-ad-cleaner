Create a Project

Prerequisites

You need to register and log in to the CAD Viewer to create your own projects. There is a limit on the number of projects you can create. Ordinary members can create up to 1 projects, Premium members can create up to 2 projects, Super members can create up to 2 projects.

Ceate a Project

Projects can be created at any membership level. You can create projects in two ways:

1. Create a project on the project list page

[Step 1]
Click "Project" in the navigation to enter the project list.

[Step 2]
Click the "+" icon button in the upper-right corner of the project list page, and select "Create Project" in the pop-up menu.

[Step 3]
Enter the name of the project in the pop-up box and click "Create Now" button to finish creating the project.

2. Create a project in the chat details of Private Chat

[Step 1]
Click "Project" in the navigation to enter the project list, and tap the icon in the upper right corner to enter the message list.

[Step 2]
In the message list page, select a friend and tap it to enter the chat details page of the friend.

[Step 3]
Click the "More" icon in the upper right corner of the chat details page with your friend to enter the chat details page with your friend.

[Step 4]
Tap the "+" icon behind your friend's avatar on the chat details screen.

[Step 5]
Enter the name of the project in the pop-up box and click "Create Now" button to finish creating the project.
