Send drawing to a project

Prerequisites

You need to have friends or projects (self-created projects and joined projects) with enough space to store drawings.

Send drawing to a project

Only by sending the drawing to the project can be viewed by other members in the project and they can communicate remotely. If you send a drawing to an ordinary project, it will occupy the space of the person in charge. If you send a drawing to a private chat friend, it will occupy the space of the sender.

1. Send drawing in a chat session

Click "Project" --> Select Project --> "Chat Details" and click "+" sign in the lower left corner and click "Drawing" in the pop-up menu. Select the drawing and send.

Note: After successful sending, the drawing will be displayed in the chat log, and can be seen and used by all members of the project. If you want to manage the drawings in the project or private chat, please check "Drawing Management". If the drawing has been renamed, it can't be sent.

2. Click "DwgChat" in drawing details to send drawings.

View the details of the drawing through the homepage, cloud map list, local map list, and project list. Click "DwgChat" on the details page, select a project or a friend, and you can send this drawing to the corresponding project or the project of the friend you chat with. At the same time, you will enter ShareView.

Note: If the drawing is already in the current project or the private chat project, you can directly go to the ShareView and open the drawing. If not, please send the drawing to the corresponding project or the private chat drawing management, and then go to the ShareView and open the drawing.

3. Click "DwgChat" on the opening page to send the drawing.

Click "DwgChat" in the drawing page, select a project or a friend, you can send the drawing to the corresponding project or private chat project. Meanwile you will enter the ShareView interface.

Note: If the drawing is already in the current project or the private chat project, you can directly go to the ShareView and open the drawing. If not, please send the drawing to the corresponding project or the private chat drawing management, and then go to the ShareView and open the drawing.
