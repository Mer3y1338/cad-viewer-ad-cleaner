Exit Project

Prerequisites

You need to join a project, and only the members in the project can use this function.

Exit Project

[Step 1]
Click "Project" --> Select Project --> "Project Details" --> "More" in the upper right corner to enter the project settings page.

[Step 2]
Click the "Exit Project" button on the project settings page to bring up a prompt for exiting the project, select OK to exit the project.

Note:After a member exits a project, the drawings uploaded by the member will belong to the person in charge. The member will not be able to access the project again.
