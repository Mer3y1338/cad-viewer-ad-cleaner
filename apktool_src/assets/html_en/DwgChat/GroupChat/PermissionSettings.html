Project Member Permission Settings

Prerequisites

You need to have created the project by creating the project. The project member permission setting function is only available to the project creator (person in charge).

Project Member Permission Settings

[Step 1]
Click "Project" to enter the project list, select a project.

[Step 2]
After entering the project, switch to the "Project Members" Tab and click on the member for which you want to set permissions.

[Step 3]
In the pop-up permission settings menu, select the permissions set for the member. The member permissions are set to view permissions by default unless modified.

Note:Only the person in charge can set the permissions for project members.
