Delete Project Members

Prerequisites

You need to have created a project by creating a project. This function is only available to the creator (the person in charge).

Delete Project Members

[Step 1]
Click "Project" to enter the project list, select a project.

[Step 2]
After entering the project, switch to the "Project Members" tab and click the "Delete Members" button to enter the page.

[Step 3]
On the Delete Project Members page, select the members to be deleted and click "Finish" to complete deleting the project members.

Note:After deleting a member, the drawings uploaded by the deleted member will belong to the responsible person, and you will not be able to enter the project again.
