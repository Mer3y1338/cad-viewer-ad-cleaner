Transfer of Project Management Rights

Prerequisites

You need to have created a project by creating a project. In CAD Viewer, project management right transfer function can only be used by the project creator (person in charge).

Transfer of Project Management Rights

[Step 1]
Click "Project" --> Select Project --> "Project Details" --> "More" in the upper right corner to enter the project settings page.

[Step 2]
Select "Transfer of project management rights" on the project settings page to enter the page for selecting a new responsible person.

[Step 3]
After selecting the new responsible person on the Select New Responsible Person page, click the Finish button to complete the transfer of project management rights.

Note: Ordinary project manager can transfer the manager status to any member of the project. After the transfer, all the drawing files in the project will be moved to the cloud storage space of the transferee, the current person in charge will become a member, and the transferee will become the new person in charge.

Note: After transferring all drawings in an ordinary project to the transferee's cloud diagram space, the transferee's cloud diagram must have available space. The number of projects created by the transferee after the transfer must not exceed the maximum limit, please refer to Creating Projects for specific limitations.
