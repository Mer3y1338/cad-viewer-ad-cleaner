Disband Project

Prerequisites

Firstly you need to have created the project. For ordinary projects, only the project creator (person in charge) could use "Disband Project".

Disband Project

[Step 1]
Click "Project" --> Select Project --> "Project Details" --> "More" in the upper right corner to enter the project settings page.

[Step 2]
Click the "Disband Project" button on the project settings page and will popup a note.

[Step 3]
Select the appropriate dissolution method as needed.

Note:
Description: Disband after backing up drawings: The system will save the latest drawings from the project to the root directory of your cloud storage space.

Direct dissolution: All drawings will be deleted and your cloud storage space will be released.

Note: It is recommended to choose to back up the drawings before dissolving the ordinary project.
