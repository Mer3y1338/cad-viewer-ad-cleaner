My Alias in project

Prerequisites

You need to have created a project or have been added to a project. Both the project creator (the person in charge) and the members in the project can use this function to change the alias in the project.

My Alias in project

[Step 1]
Click "Project" --> Select Project --> "Project Details" --> "More" in the upper right corner to enter the project settings page.

[Step 2]
Click the "My Alias in project" section on the project settings screen to access

[Step 3]
Enter the name on "My Alias in Project" and click "Done".

Note: The modified name will be seen by others in the project, so you can set the name in multiple projects separately.
