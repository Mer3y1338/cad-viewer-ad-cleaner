Project Settings

Prerequisites

You need to create a project by creating a project or have already joined a project. The project creator (responsible person) and members within the project can use the project

Project Settings

The project settings are different for the creator (person in charge) and the project members (joiners):

1. Project settings for "responsible person"
Click "Project" --> Select Project --> "Project Details" --> "More" to enter the project settings page. The person in charge can use the following functions: Modify project name, Project properties, Transfer Ownership, Put project on top, Project space usage, My Alias in Project, My permissions, Search chat
Modify Project Name、 Project Properties、 Transfer of Project Management Rights、 Put project on top、 Project Space Usage、 My Alias in Project、 My permissions、 Search chat history、 Disband project

2. Project settings for Member
Click "Project" --> Select Project --> "Project Details" --> "More" in the upper right corner to enter the project settings page. Project members can use the following functions: Add Member to Project, Project Properties, Put Project on Top, Project Space Usage, My Alias in Project, My Permissions, Search Chat History, Quit Project"
Add members into the project、 Project Properties、 Put project on top、 Project space usage、 My Alias in Project、 My permissions、 Search chat history、 Disband project
