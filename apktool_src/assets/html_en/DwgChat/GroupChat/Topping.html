Put Project on Top

Prerequisites

You need to have created a project or have been added to a project.Both the project creator (the person in charge) and the members in the project can use this function to put project on top.

Put Project on Top

[Step 1]
Click "Project" --> Select Project --> "Project Details" --> "More" in the upper right corner to enter the project settings page.

[Step 2]
Click the "Put Project on Top" to turn on.

Note:Items that are topped will be at the top of the list of items, with post-topped items at the top of the list.
