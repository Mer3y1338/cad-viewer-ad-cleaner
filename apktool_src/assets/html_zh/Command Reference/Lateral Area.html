测立面

测量 → 测立面

功能描述：模板面积是建筑的侧面积，在CAD看图王中将测量侧面积称为测立面。 计算侧面积时需要知道两个量，一个是总长，一个是高度，二者的乘积为侧面积值。

操作步骤：
步骤1：在“测量”功能中点击“测立面”启动测量侧面积的命令；
步骤2：指定总长值：
总长值可以通过测量图纸上的点，汇总得到总长；也可以直接在总长输入框中输入总长；
步骤3：指定高度值：
高度值不直接显示在图纸上，所以需要手动输入。
步骤4：计算侧面积值：
获得总长和高度值后，点击“计算”按钮即可得到侧面积值。
步骤5：面板上方有“勾选后把结果标在图上”的勾选框，如勾选，测量完成时，侧面积结果标注在图上；如未勾选，测量完成时，图纸上不显示测量结果。

测量面板功能说明：
1. 面板上方有“勾选后把结果标在图上”的勾选框，如勾选，测量完成时，面积结果标注在图上；如未勾选，测量完成时，图纸上不显示面积结果。勾选后，在测量过程中，可以给每个侧面积值命名。
2. 点击“设比例”图标，支持跳转到比例列表页面，可修改测量比例。
3. 点击“颜色”图标，可修改长度标注的显示颜色；
4. 点击“字高设置”图标，支持修改字高。字高默认为自适应屏幕，可切换为自定义字高。自定义字高时，支持手动输入字高值，也可通过“字高匹配”功能将字高匹配为图纸中指定文字的字高。
5. 点击面板上的“√”为确认当前的点并结束命令。
6. 点击面板上的“×”或命令提示语后的“×”关键字为放弃当前的点并结束命令。

Tips：
（1）点击“设置”，支持将测量的面积结果以米（m）单位显示；可在结果框内长按，选择“复制”复制测量结果。
（2）记录结果后，可在“看结果”中查看。
（3）对于标注在图上的结果，可以在“结果统计”中查看并导出到Excel中。
