线型设置

线型设置：设置新绘制对象的线型。

切换当前线型的操作：

操作步骤：

步骤1：在“设置”中，点击“线型设置”；

步骤2：在“线型设置”弹框中选择需要的线型。

步骤3：点击“确定”即可。

从程序自带的线型文件中选择线型添加到当前图纸：

操作步骤：

步骤1：在“设置”中，点击“线型设置”；

步骤2：在“线型设置”弹框中，点击右上角的“+”；

步骤3：从列表中选择要添加的线型；

步骤4：点击“确定”即可看到选择的线型已添加到当前图纸的线型列表中。

从外部线型文件中选择线型添加到当前图纸

操作步骤：

步骤1：在“设置”中，点击“线型设置”；

步骤2：在“线型设置”弹框中，点击右上角的“+”；

步骤3：点击底部的“选择线型文件”按钮；

步骤4：浏览设备中存放线型文件的路径，点击选择需要的线型文件；

步骤5：在展示的线型列表中选择需要的线型

步骤5：点击“确定”即可看到选择的线型已添加到当前图纸的线型列表中。
