测长度

测量 → 测长度

操作步骤：

单击工具条上的“测量”按钮，弹出的面板上选择“测长度”命令，默认为单段测量模式，点击面板上的模式切换开关后可切换到连续测量模式。

单段测量模式下：
步骤1：点击或拖动指定一点作为测量长度的起点。
步骤2：点击或拖动指定第二点作为测量长度的终点。指定完终点后，可拖动圆形标记调整起点和终点的位置。面板上显示该线段的长度，角度，X轴增量，Y轴增量。
步骤3：点击更多点，可测更多长度。

连续测量模式：
步骤1：点击或拖动指定一点作为测量长度的起点；
步骤2：点击或拖动指定第二点；
步骤3：可继续点击或拖动指定第三点或更多点，面板上显示各条线段的总长。测量过程中，可拖动圆形点标记调整线段的长度。

测量面板功能说明：
1. 面板上方有“勾选后把结果标在图上”的勾选框，如勾选，测量完成时，长度结果标注在图上；如未勾选，测量完成时，图纸上不显示长度结果。
2. 点击“设比例”图标，支持跳转到比例列表页面，可修改测量比例。
3. 点击“精度”图标，支持跳转到精度设置页面，可修改图纸精度设置。
4. 点击“颜色”图标，可修改长度标注的显示颜色；
5. 点击“字高设置”图标，支持修改字高。字高默认为自适应屏幕，可切换为自定义字高。自定义字高时，支持手动输入字高值，也可通过“字高匹配”功能将字高匹配为图纸中指定文字的字高。
6. 点击面板上的“√”为确认当前的点并结束命令。
7. 点击面板上的“×”或命令提示语后的“×”关键字为放弃当前的点并结束命令。

其他说明：
1. 记录结果后，可在“看结果”中导出“详细结果”查看各段长度。
2. 对于标注在图上的结果，可以在“结果统计”中查看并导出到Excel中。
3. 如已经掌握了手指长按屏幕后精确取点的方法，可以在设置里关闭“测量后支持调整点位置”，简化测量流程。
