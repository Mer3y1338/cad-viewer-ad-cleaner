设比例

初始状态有三个比例：1：1，1：50，1：100。

在模型空间测量时，默认比例为1:1。
在布局空间测量时，若通过精确捕捉获取的点均在视口内，则测量比例为视口比例；若获取的点在视口外则比例为1:1。

置为当前比例：
点击比例列表中的某个比例则将该比例置为当前比例。

删除比例：
可删除列表中已有比例。长按后显示“删除”，点击后立即删除。

添加比例：
点击“测量比例”右上角的+，输入要添加的比例，点击“确定”即可添加比例。

若不确定需要添加的比例是多少，可以从图纸中量取一段长度指定比例，点击“量取一段长度指定比例>“用测长度的方法指定两点即可。

指定两点后，会在弹框中显示测量长度值，输入期望长度值后，即可自动计算出新比例值（新比例值=测量长度/期望长度）。


点击“确定”即可把该比例添加到比例列表中。
