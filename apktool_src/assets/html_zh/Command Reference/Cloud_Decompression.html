云解压

功能说明：

云图的压缩包支持本地解压和云解压两种解压方式。本地解压是指把云图的压缩包先下载到手机的内部存储里，然后把解压文件存储在手机里。云解压是指在云端解压，把解压文件存储在云图目录下。

操作步骤：

步骤1：在云图列表里，点击一个压缩包文件；

步骤2：在提示弹框里点击底部的“云解压”按钮，则在后台自动解压，解压成功后可在压缩包同目录下查看到解压文件；

步骤3：点击界面右上角的云解压进度图标查看解压进度；

在云解压列表中，支持以下功能：

1. 查看转换状态，云解压任务的状态包括解压中，解压成功，解压失败。

2. 在列表里的某一个解压文件上长按，支持将该解压记录清除。

3. 已转换成功的压缩包，显示解压的位置，点击整条记录可跳转到解压的位置。

4. 点击右上角的“批量操作”按钮，可以批量清理云解压记录。
