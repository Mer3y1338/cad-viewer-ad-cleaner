注释

绘图 → 注释

操作步骤：

步骤1：点击“绘图”按钮，选择“注释”命令。

步骤2：单击或拖动指定引线的起点。

步骤3：调整起点或指定引线终点。

步骤4：在弹出的输入框中填写注释文字。
说明：点击“字高设置”图标，支持修改字高。 字高默认为自适应屏幕，可切换为自定义字高。自定义字高时，支持手动输入字高值，也可通过“字高匹配”功能将字高匹配为图纸中指定文字的字高。

步骤 5：点击确定按钮完成注释。

编辑注释：
创建完成的注释对象，可选中后进行编辑。选中箭头和引线，点击“夹点编辑”图标，可调整引线的方向和大小；选中文字，点击编辑菜单中的“修改文字”选项可修改文字内容。
