测面积

测量 → 测面积

功能描述：测面积功能支持测量以直线段或圆弧段为边界的封闭区域的面积。测量时默认两点之间的连线是直线，可以通过点击命令提示下方的“圆弧”关键字切换到选择圆弧的模式。

操作步骤：
步骤1：点击工具条上的“测量”按钮，弹出的面板上选择“测面积”命令。
步骤2：点击或拖动指定第一点。
步骤3：单击或拖动指定测量面积的下一点。可点击“+”放弃当前面积测量，并开始新的面积测量，点击“+”后，回到步骤2。
步骤4：单击或拖动指定测量面积的下一点。面板中显示闭合多段线所围区域的面积和周长。可点击“+”放弃当前面积测量，并开始新的面积测量。
步骤5：单击或拖动指定测量面积的下一点。同步骤4。

测量面板功能说明：
1. 面板上方有“勾选后把结果标在图上”的勾选框，如勾选，测量完成时，面积结果标注在图上；如未勾选，测量完成时，图纸上不显示面积结果。勾选后，在测量过程中，可以给每个面积值命名。
2. 点击“设比例”图标，支持跳转到比例列表页面，可修改测量比例。
3. 点击“颜色”图标，可修改长度标注的显示颜色；
4. 点击“字高设置”图标，支持修改字高。字高默认为自适应屏幕，可切换为自定义字高。自定义字高时，支持手动输入字高值，也可通过“字高匹配”功能将字高匹配为图纸中指定文字的字高。
5. 点击面板上的“√”为确认当前的点并结束命令。
6. 点击面板上的“×”或命令提示语后的“×”关键字为放弃当前的点并结束命令。

Tips：
（1）“设置”，支持将测量的面积结果以米（m）单位显示；可在结果框内长按，选择“复制”复制测量结果。
（2）记录结果后，可在“看结果”中查看。
（3）对于标注在图上的结果，可以在“结果统计”中查看并导出到Excel中。
