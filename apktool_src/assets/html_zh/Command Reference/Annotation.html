批注命令

批注功能包含手绘线，箭头，文字，云线，音频，图片，视频，引线，直线，矩形，椭圆，共11种批注类型，支持批注的文字和字母编号功能。
在CAD看图王中，新创建的批注对象是独立于图纸但又和图纸相关联的特殊实体，无需执行存图操作即可自动保存。在本地，如果移动图纸，重命名图纸，或者通过QQ，微信等分享给好友，可能图纸中的批注因失去关联不再显示。以下是使用批注的小技巧：

Tip 1： 建议先将图纸上传到云图上，然后从云图上打开该图纸再添加批注，这样能确保添加的批注数据不丢失。

比如，我想在A图纸里添加批注，可按如下步骤操作：
Step 1： 首先在A图纸的“文件详情”页里，点击“上传”，选择“云图”,指定路径，点击“确定”，即可添加到我的云图账号中；
Step 2： 点击底部导航栏中的“云图”，点击打开刚才上传的A图纸，选择开图模式，然后就可以添加批注了。
Step 3： 添加完批注后，无需保存图纸即可直接退出，退出后A图纸的文件状态将由绿色变成红色，点击红色按钮开始同步，A图纸的文件状态从红色变成绿色表示同步完成。

在云图中打开文件，添加并同步批注，能从根本上避免批注数据丢失。

Tip 2： 添加图片批注时，能添加几张图片？

最多支持添加10张图片。

Tip 3： 添加音频批注时，最长能录制多长的语音？

最长不要超过60s。

Tip 4： 能修改已经添加的批注吗？

当然可以，点击要修改的批注对象，会在下方显示编辑批注的工具栏，不同的批注类型支持修改的属性也不同，通常支持删除，修改颜色的操作，有些对象还支持修改线宽，或修改文字内容等操作。

Tip 5： 我想在手机和pad上都能查看添加了批注的图纸文件，怎么操作？

按照Tip 1中的操作，首先将图纸上传到云图里，然后在手机和pad上登录同一个云图账号，即可查看了。

Tip 6： 我想把添加了批注的图纸文件分享给我的好友，怎么操作？

按照Tip 1中的操作，首先将图纸上传到云图里，然后在云图中长按该文件名，弹出的菜单中点击“分享”，设置时效和可见性后即可选择微信，或QQ等方式分享了。
