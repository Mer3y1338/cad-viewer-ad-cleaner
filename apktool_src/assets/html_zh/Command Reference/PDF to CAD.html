PDF 转 CAD

操作步骤：

步骤1：打开一张PDF文件；

步骤2：点击底部的“PDF转CAD”按钮，则自动上传并执行转换；

步骤3：点击界面右上角的图标查看转换进度；或点击首页 > + > PDF转CAD，查看转换进度。转换完成后生成的DWG文件存放在用户的云图 > PDF转CAD目录下，可进入该目录点击对应的文件打开查看。

在PDF转CAD列表中，支持以下功能：

1. 查看转换状态，PDF任务的转换状态包括转换中，转换成功，转换失败；

2. 支持跳转到生成的CAD文件的保存位置：点击CAD文件保存位置尾部的“打开”按钮支持跳转到云图 > PDF转CAD目录下；

3. 点击转换列表中的某一个PDF尾部的箭头，会弹出提示告知转换结果，在有转换结果的弹框里点击“立即查看”，支持跳转到云图目录下生产的CAD文件对应位置。

4. 在列表里的某一个PDF文件上长按，支持将该PDF转换记录清除。

5. 点击底部的“添加PDF”按钮，从最近列表，本地图纸列表，和云图图纸列表里选择一个或多个PDF，并将其添加到转换任务列表里。

6. 点击右上角的“批量操作”按钮，可以批量清理PDF转CAD的转换记录。
