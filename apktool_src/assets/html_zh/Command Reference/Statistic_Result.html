结果统计

测量 → 结果统计

功能描述：在测量过程中，使用标结果的功能，把长度值，面积值，或侧面积值标在图上后，可以使用&ldquo;结果统计&rdquo;功能分类查看，并支持分类导出到Excel中。

操作步骤：
步骤1：点击工具条上的“测量”按钮，弹出的面板上选择“结果统计”命令。
步骤2：指定统计范围，默认统计图纸中的全部测量结果，默认展示全部颜色的测量结果，可以框选指定统计范围（支持矩形窗口和异形窗口范围），或指定颜色。
步骤3：在列表中展示符合筛选条件的统计结果。
步骤4：点击“导出”可将当前类型的测量结果导出到Excel表格中。

说明：在面积和侧面积结果列表里，点击“名称”可修改该测量结果的名称。
