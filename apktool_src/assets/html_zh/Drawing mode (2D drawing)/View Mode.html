看图模式界面及功能介绍

顶部功能：

返回：退出当前图纸。

多图查看：支持同时打开并查看两张图纸，上下对比查看。

全图：显示全图。

全屏：显示全屏。

更多：点击可选择编辑模式，文件信息，导出，帮助，下载批注资源，隐藏批注，重生成，和设置选项。

侧边功能：

共享聊图：进入共享聊图。

底部功能：

快捷命令：展示最近使用的命令图标，点击快捷命令可快速启动该命令，等同于点击菜单栏中的该命令。快捷命令默认为显示，可在“设置”中关闭。长按快捷命令上的命令图标可以将其固定在右侧显示，变成常驻命令。在常驻命令上长按，也可取消常驻。

批注：包含手绘线，箭头，文字，云线，音频，图片，视频，引线，直线，矩形，椭圆批注功能，并支持基于批注文字自动编号。

图层：包括图层列表，关闭图层，关闭其他，上一图层，和全部打开。

测量：包括测长度，连续测，测面积，测立面，测坐标，测弧长，测实体，测角度，设比例，看结果，结果计统，和精度。

布局：切换任意模型或布局空间。

工具：提供文字查找，块统计，和书签功能。

视觉样式：支持二维和三维视觉样式切换。
